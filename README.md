# Communication Réseau Sécurisée

Ce projet est une application Android démontrant l'implémentation de diverses techniques de sécurisation des communications réseau.

## Fonctionnalités
- Support du protocole HTTP/3 (QUIC) via OkHttp.
- Communication bidirectionnelle sécurisée avec WebSockets.
- Authentification mutuelle via Mutual TLS (mTLS).
- Résolution DNS sécurisée avec DNS-over-HTTPS (DoH).
- Vérification de la validité des certificats via Certificate Transparency.

## Réalisation
Réalisé par Ibtissam.
